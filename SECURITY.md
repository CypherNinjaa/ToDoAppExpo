# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1. **DO NOT** open a public issue
2. Email details to: vikashkelly@gmail.com
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

## What to Expect

- **Response Time:** Within 48 hours
- **Updates:** Regular updates on the progress
- **Resolution:** We'll work with you to resolve the issue
- **Credit:** We'll acknowledge your contribution (if desired)

## Security Best Practices

When contributing code, please ensure:

- No hardcoded credentials or API keys
- Proper input validation
- Secure data storage practices
- Following React Native security guidelines

Thank you for helping keep TaskShell secure!
